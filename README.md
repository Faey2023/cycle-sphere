## Getting Started

1. **Clone the repository:**

   ```bash
   git clone https://github.com/Faey2023/cycle-sphere

   ```

2. **Navigate to the project directory:**

   ```bash
   cd cycle-sphere
   ```

3. **Install dependencies:**

   ```bash
   npm install
   ```

4. **Run the project:**
   ```bash
   npm run dev
   ```
